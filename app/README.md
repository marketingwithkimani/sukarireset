# Sukari Reset Companion App

The interactive mobile application for the Sukari Reset ecosystem.

## Features
- **Habari Storytelling**: Daily morning inspiration.
- **Smart Logger**: Local food tracking (Ugali, Sukuma, etc.)
- **Jamii Community**: Private circles for peer support.
- **Wellness Pillars**: Fasting, Movement, and Kula Vizuri principles.

## Tech Stack
- React 19 + Vite
- Tailwind CSS 4
- GSAP for high-fidelity transitions
- Lucide React Icons

## Development
```bash
npm install
npm run dev
```

## Production Build
```bash
npm run build
```
The build output in `dist/` is ready to be wrapped as a PWA or mobile application using Capacitor/Cordova.
